# Postictal Pilot: LGS After-Visit Guide

This is a draft patient-family education pilot for Lennox-Gastaut syndrome after clinic counseling.

## Review Status

- Draft module for team review.
- Not ready for patient-facing distribution.
- Clinician/source review is still required.
- Audio is a short NotebookLM draft, not a final reviewed recording.

## Included

- `index.html` - low-text visual webpage.
- `assets/` - illustration and embedded audio.
- `docs/source-packet.md` - source trail and generation log.
- `docs/notebooklm-source-packet.md` - audio steering packet.
- `docs/clinician-review-checklist.md` - review checklist.

## Review Questions

- Does the medication/device framing feel clinically safe and useful?
- Is the distinction between daily medicines and rescue medicines clear?
- Are VNS, RNS, DBS, corpus callosotomy, diet therapy, and surgery evaluation framed conservatively enough?
- Should this pilot be clinic-neutral or Emory-branded before sharing with patients?
