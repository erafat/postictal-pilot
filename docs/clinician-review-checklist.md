# LGS Module - Clinician Review Checklist

## Clinical Accuracy
- [ ] LGS definition is accurate and not oversimplified into one seizure type.
- [ ] Seizure type descriptions are plain-language but not misleading.
- [ ] Medication examples are framed as examples, not recommendations.
- [ ] Diet, device, and surgery-oriented options are framed as discussion categories, not promises.
- [ ] Rescue medicine language defers to the prescribed seizure action plan.
- [ ] Emergency-care language does not conflict with local clinic instructions or the patient's own plan.

## Patient-Facing Clarity
- [ ] The page is understandable for a caregiver without medical training.
- [ ] The page does not feel like a pamphlet.
- [ ] The first screen tells families what to do with the information.
- [ ] The page explains what to track before the next visit.
- [ ] The page separates daily plan, rescue plan, and future options.

## Safety / Side Effects
- [ ] No PHI or patient-identifiable anecdotes.
- [ ] No individualized diagnosis or medication advice.
- [ ] No language implying replacement of clinical care.
- [ ] No unsupported local Emory process instructions.

## Share Readiness
- [ ] Source packet reviewed.
- [ ] Audio script reviewed.
- [ ] Visual page reviewed on mobile.
- [ ] Final disclaimer reviewed.
- [ ] Resource links checked.
